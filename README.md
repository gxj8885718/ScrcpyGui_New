# ScrcpyGui_New
another scrcpygui written by AutoHotKey

这是一个使用AutoHotKey重写的scrcpygui，意在提供轻量化、功能化、图形化地操纵scrcpy工具。

其最主要的目的其实是......隐藏直接运行scrcpy程序的命令行窗口。
